# 🌍 AI Language Translation Tool

## 📌 Project Overview

The AI Language Translation Tool is a web-based application developed as part of the CodeAlpha Artificial Intelligence Internship.

The application allows users to translate text between multiple languages using the MyMemory Translation API. It provides a clean and responsive interface along with several useful features to improve the user experience.

## 🚀 Features

- Translate text into multiple languages
- Language Swap
- Voice Input using Speech Recognition
- Text-to-Speech for translated text
- Copy translated text
- Dark Mode / Light Mode
- Translation History using Local Storage
- Clear Translation History
- Loading Indicator while translating
- Responsive Design for Mobile and Desktop

## ⚙️ How to Run

1. Download or Clone the repository.
2. Open the project folder.
3. Open **index.html** in your browser.
4. Enter text.
5. Select source and target language.
6. Click **Translate**.
